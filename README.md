# mariokart.info
[mariokart.info](https://mariokart.info) is a web app that allows for finding, optimizing and comparing builds for Mario Kart World, Mario Kart 8 Deluxe, Mario Kart 8, and Mario Kart 7.

![Screenshot of the mariokart.info interface.](https://mariokart.info/resources/graphics/github-thumbnail.png)


## Contribute
[mariokart.info](https://mariokart.info) is an artisanal website crafted with care. The only dependency you’ll need to contribute is [Sass](https://sass-lang.com).

All contributions are welcome, whether it be code, [bug reports](https://github.com/JeremieD/mariokart.info/issues), or suggestions.
