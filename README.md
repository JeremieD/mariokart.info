# mariokart.info
mariokart.info is a web app that allows for finding, optimizing and comparing builds for Mario Kart X, Mario Kart 8 Deluxe, Mario Kart 8, and Mario Kart 7!

![Screenshot of the mariokart.info interface.](https://mariokart.info/resources/graphics/github-thumbnail.png)


## Contribute
All contributions are welcome, whether it be code, [bug reports](https://github.com/JeremieD/mariokart.info/issues), or suggestions.

mariokart.info is a super simple static website. No JS libraries are used.

The only thing you'll need is [Sass](https://sass-lang.com) to contribute CSS.
